# خزنة كلمات المرور (Password Vault)

منصة عربية آمنة لحفظ معلومات الدخول للتطبيقات والمواقع.

## المميزات

- ✅ واجهة عربية RTL
- ✅ تشفير AES-256 لكلمات المرور
- ✅ دعم PWA للتحويل إلى تطبيق
- ✅ رفع ملفات (PDF, صور, نصوص)
- ✅ Docker للنشر السهل

## التشغيل

### باستخدام Docker (موصى به)

```bash
docker-compose up --build
```

الموقع سيعمل على: http://localhost:3000

### التشغيل المحلي

1. تثبيت الاعتماديات:
```bash
npm install
```

2. إعداد قاعدة البيانات:
```bash
npx prisma db push
```

3. تشغيل التطبيق:
```bash
npm run dev
```

## متغيرات البيئة

| المتغير | الوصف |
|---------|-------|
| `DATABASE_URL` | رابط PostgreSQL |
| `ENCRYPTION_KEY` | مفتاح التشفير (32 حرف) |
| `JWT_SECRET` | مفتاح JWT |

## الأمان

- كلمات المرور المحفوظة مشفرة بـ AES-256-GCM
- كلمات مرور المستخدمين مهشّرة بـ bcrypt
- JWT للجلسات مع انتهاء صلاحية 7 أيام
